usage:

##################################

see also: https://github.com/chessstats/players

create directory called 'current' in the working directory
copy the players.txt file created with the players package into this directory

then at the R console type:

library(procplayers)
process_players()

this will create statistical files from players.txt in the 'current/stats' directory
